# What We Observe

- Percentage of steps completed per session
- Which step TYPE causes most drop-offs (text, MCQ, exercise, reflection)
- Time spent per step type
- Whether user returns to finish incomplete sessions
- Step index where drop-off typically occurs

# What It Tells Us

| Pattern | Interpretation |
|---------|---------------|
| Completes all steps, fast pace | Auto-Pilot — dutiful but possibly not deep processing |
| Drops off at exercises | May find exercises uncomfortable — needs gentler framing |
| Drops off at long text blocks | Content length issue — shorten intro variants for this user |
| Completes exercises, skips MCQ | Experiential learner — prefer hands-on content |
| Returns next day to finish | Highly engaged but time-constrained — respect their pace |
| Never returns to incomplete sessions | Content friction — simplify or shorten |

# How It Influences Personalization

1. **Preferred step types → content weighting**: If user engages most with exercises, lead with exercises
2. **Typical dropout step → intervention**: If they always drop at step 5, make step 5 shorter or more engaging
3. **Fast completers → depth variants**: Offer "dig deeper" optional content
4. **Slow completers → patience**: Never rush; remove "estimated time" pressure
5. **Low completion + low cravings**: Possible low engagement risk — increase notification frequency
