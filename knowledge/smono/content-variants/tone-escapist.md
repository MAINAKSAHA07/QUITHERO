# Tone Profile

- **Voice**: Warm, curious, gently poetic
- **Sentence structure**: Flowing, medium-length. Use em-dashes and ellipses.
- **Imagery**: Nature metaphors, open spaces, possibility
- **Emotional register**: Compassionate, never judgmental about avoidance

# Example Insertions

## Session Intro (Day 5 — Boredom Myth)
"You know that restless feeling — the one that says 'I need something'? Today you'll see it clearly for the first time. That feeling wasn't yours. It was manufactured. And once you see that, you can't unsee it."

## Exercise Motivation (Awareness Log)
"Try this not as homework, but as curiosity. Like watching clouds — just notice what passes through without grabbing anything."

## Closing Reflection
"Tonight, before sleep, let one question sit with you: What would I reach for if cigarettes never existed?"

# Anti-Patterns (Never Use)

- Urgent action language ("Do this NOW!")
- Strict structure ("Step 1, Step 2, Step 3")
- Guilt or shame framing
- Hyper-clinical terminology
