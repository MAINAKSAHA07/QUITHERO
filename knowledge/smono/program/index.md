# Program Overview

The Smono program runs 30 days. Days 1–9 are preparation (the user continues smoking). Day 10 is the quit moment. Days 11–30 are freedom reinforcement.

## Program Phases

| Phase | Days | Focus |
|-------|------|-------|
| Understanding | 1–5 | See the trap: psychoeducation, myths, brainwashing |
| Preparation | 6–9 | Dismantle beliefs, prepare mentally for freedom |
| Reset | 10 | The quit moment — reframed as liberation, not sacrifice |
| Early Freedom | 11–15 | Navigate triggers, social pressure, the "just one" myth |
| Consolidation | 16–22 | Emotional resilience, stress tools, body recovery |
| Lifelong Freedom | 23–30 | Complacency guards, high-risk plans, identity shift |

## CBT Techniques Used

- Psychoeducation (Days 1–3)
- Functional analysis — A→B→C model (Day 1)
- Cognitive restructuring (Days 4–8)
- Behavioral experiments (Days 11–15)
- Exposure with response prevention (Days 12–14)
- Stress inoculation (Day 17)
- Relapse prevention planning (Days 23–25)
- Values clarification (Day 29)
